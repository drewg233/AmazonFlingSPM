# Amazon Fling Swift Package

## Installation
- Add the following dependency to your project's `Package.swift`.

```swift
dependencies: [
    .package(url: "https://github.com/mskrischke/AmazonFlingSPM.git", .upToNextMinor(from: "1.4.0"))
]
```
